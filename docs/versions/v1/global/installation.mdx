<Info>
  **Prerequisite**: Please install Node.js (version 19 or higher) before
  proceeding.
</Info>

Follow these steps to install and run Dev.to Rater on your operating system:

**Step 1**: Clone repository:

<CodeGroup>

```bash bash
git clone https://github.com/perisicnikola37/dev-to-rater
```

</CodeGroup>

**Step 2**: Install dependencies

Navigate to the project directory (where the `package.json` file is located) and execute the following command:

```bash bash
npm install
```

**Step 3**: Run the application

Once the installation is complete, start the application with the following command:

```bash bash
npm run dev
```

You can now access a local preview of the application at: `http://localhost:5173`.

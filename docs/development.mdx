---
title: "Installation"
---

<Info>
  **Prerequisite**: Please install Node.js (version 19 or higher) before
  proceeding.
</Info>

Follow these steps to install and run Dev.to Rater on your operating system:

**Step 1**: Clone repository:

<CodeGroup>

```bash git
git clone https://github.com/perisicnikola37/dev-to-rater
```

</CodeGroup>

**Step 2**: Navigate to the project directory (where the `package.json` file is located) and execute the following command:

```bash npm
npm install | npm i
```

**Step 3**: Run the application using the following command:

```bash npm
npm run dev
```

A local preview of the application will be available at `http://localhost:5173`.
